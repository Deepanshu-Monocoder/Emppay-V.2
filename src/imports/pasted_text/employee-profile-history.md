Design a modern, professional Employee Payroll & Attendance Management System for a real business. The application already contains Dashboard, Payroll, Employees, Holidays, and Settings modules. The new feature to design is a comprehensive Employee Profile & History Center.

IMPORTANT:
- Must be fully responsive for Desktop, Tablet, and Mobile.
- Desktop uses a left sidebar navigation.
- Mobile uses bottom navigation.
- Focus on usability for business owners, HR staff, and payroll managers.
- Clean SaaS-style interface.
- Modern cards, tables, filters, and analytics.
- Efficient workflows with minimal clicks.

--------------------------------------------------
EMPLOYEE PROFILE & HISTORY CENTER
--------------------------------------------------

When a user clicks an employee from the Employees page, open a dedicated Employee Profile page.

Desktop:
- Full-page profile view.
- Easy navigation back to employee list.

Mobile:
- Separate profile screen.
- Optimized touch interactions.

--------------------------------------------------
HEADER SECTION
--------------------------------------------------

Display:

- Employee Avatar/Photo
- Employee Name
- Employee ID
- Department
- Monthly Salary
- Joining Date
- Employment Status (Active/Inactive)

Actions:

- Edit Employee
- Export Data
- Back to Employees

Desktop:
- Information and actions displayed in a clean horizontal layout.

Mobile:
- Stack information vertically with prominent action buttons.

--------------------------------------------------
DATE RANGE FILTER
--------------------------------------------------

A powerful date range filter controls all information on the page.

Fields:

- From Date
- To Date
- Apply
- Reset

Quick Filters:

- This Month
- Last Month
- Last 3 Months
- Last 6 Months
- This Year
- Entire Employment History

Examples:
Owner can view:
- January attendance only
- January to March attendance
- Entire year history
- Complete employment history
